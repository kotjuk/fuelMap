# FuelMap

Карта АЗС с ценами на топливо и пользовательскими отчётами о наличии бензина и очередях.

## Что уже работает

- Сбор АЗС Роснефти из API
- Сохранение цен в SQLite
- Автоматическое обновление цен
- Веб-интерфейс со списком АЗС
- Форма пользовательского отчёта:
  - наличие АИ-92
  - наличие АИ-95
  - наличие 95 Pulsar
  - наличие ДТ
  - очередь
  - комментарий

## Запуск локально

Сначала запустить автообновление цен:

```bash
php run_collector_loop.php
```

Во втором терминале запустить сайт:

```bash
php -S 127.0.0.1:8000 -t public
```

После этого открыть:

```text
http://127.0.0.1:8000
```

## Ручной сбор данных

```bash
php collect_rosneft.php
```

## Структура проекта

```text
fuel-map/
  collect_rosneft.php
  run_collector_loop.php
  public/
    index.html
    app.js
    api.php
    report.php
```

## База данных

Проект использует SQLite.

Файл базы:

```text
fuel.sqlite
```

Этот файл не хранится в GitHub и создаётся автоматически после запуска:

```bash
php collect_rosneft.php
```

## Важно

GitHub Pages не подходит для запуска этого проекта, потому что проект использует PHP и SQLite.

Для публикации нужен хостинг с поддержкой PHP.
